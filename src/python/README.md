# Python Services – Overview

این دایرکتوری شامل سرویس‌های Python مربوط به Face-Platform است. هر سرویس عموماً یک FastAPI (یا سرویس مشابه) است که از طریق HTTP با Face.Api (.NET) و سایر بخش‌ها صحبت می‌کند.

(توضیحات کامل قراردادها، پورت‌ها و ساختار دایرکتوری در گفت‌وگو آمده و می‌تواند در این فایل به‌مرور تکمیل شود.)
